🎯 Guesser Game

A simple Java console game where players try to guess a number chosen by a guesser.
An Umpire collects all guesses and decides the winner.

🧩 Game Roles

The game consists of four roles:

Guesser – Chooses a number between 1 and 10

Player 1

Player 2

Player 3

Umpire – Collects numbers and determines the winner

🎮 Game Rules

The Guesser selects a number between 1 and 10.

The Umpire collects the number from the guesser.

Three players each guess a number between 1 and 10.

The Umpire compares the guesser's number with the players' guesses.

The first player whose guess matches the guesser's number wins the game.

If no player guesses correctly, no one wins the round.

🛠️ Technologies Used

Java

Object Oriented Programming (OOP)

Scanner class for user input

▶️ How to Run
